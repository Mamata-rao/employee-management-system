# Employee Management System

A production-style Spring Boot project built for learning enterprise Java development.

## Tech Stack

- Java 21
- Spring Boot 3
- Spring Data JPA
- PostgreSQL
- Maven
- IntelliJ IDEA

## Current Features

- Spring Boot project setup
- PostgreSQL configuration
- REST API
- GET /employee endpoint

## Project Structure

```
src
 └── main
      ├── java
      └── resources
```

## How to Run

1. Start PostgreSQL
2. Create the database `employee_db`
3. Update `application.properties`
4. Run `EmployeeServiceApplication`
5. Open:

http://localhost:8080/employee

Expected response:

Hello Employee
